# SAP Commerce Test Helper

Simplifies running SAP Commerce (Hybris) tests in IntelliJ.

The plugin automatically prepares the required classpath and JVM environment so unit and integration tests can be executed directly from extension modules without manual configuration.

## Prerequisite

Pre-created modules using the SAP Commerce Development Toolset Plugin.
